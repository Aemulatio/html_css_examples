HTML Popover API

Нативные всплывающие панели без библиотек. Есть режимы `auto` и `manual`.

1) Auto popover

`auto` закрывается по Esc, клику вне, и при открытии другого auto-popover.

Используй для тултипов, коротких подсказок, контекстных заметок.

2) Manual popover

Это `manual`: сам управляешь show/hide/toggle.

Удобно для кастомных меню и сценариев с явным контролем состояния.

Мини-шаблон

<button popovertarget="help">?</button>
<div id="help" popover>Подсказка</div>

<button popovertarget="menu" popovertargetaction="toggle">Menu</button>
<div id="menu" popover="manual">...</div>
