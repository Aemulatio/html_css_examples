CSS Anchor Positioning

Экспериментальная фича: элемент позиционируется относительно другого элемента-якоря.

1) Подсказка под кнопкой

Блок привязан к кнопке через `position-anchor` и `anchor()`.

2) Notification badge у чипа

Inbox 3

Badge остается привязан к чипу даже при изменении размеров текста.

Мини-шаблон

.trigger { anchor-name: --t; }

.pop {
  position: absolute;
  position-anchor: --t;
  left: anchor(left);
  top: calc(anchor(bottom) + 8px);
}
