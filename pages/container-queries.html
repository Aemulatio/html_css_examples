CSS Container Queries

Меняется не от ширины окна, а от ширины контейнера. Потяни правую границу блока.

State

Карточка #1

На 520px меняется внутренняя сетка.

Query

Карточка #2

На 760px становится 3 колонки.

Layout

Карточка #3

Окно можно не трогать, важен только контейнер.

Мини-шаблон

.host {
  container-name: cards;
  container-type: inline-size;
}

@container cards (min-width: 520px) {
  .cards { grid-template-columns: 1fr 1fr; }
}

@container cards (min-width: 760px) {
  .cards { grid-template-columns: repeat(3, 1fr); }
}

Когда компонент живет в сайдбаре, модалке или карточке, `@container` обычно лучше `@media`.
