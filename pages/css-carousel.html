CSS Carousel (по MDN Overflow Guide)

Реализация через `::scroll-button()` и `::scroll-marker`, без JavaScript.

  • Слайд 1

    Flex + scroll-snap + generated controls.

  • Слайд 2

    Кнопки prev/next генерируются браузером.

  • Слайд 3

    Маркеры пагинации тоже нативные.

  • Слайд 4

    Активный marker выделяется через :target-current.

Если браузер пока не поддерживает CSS Carousel pseudo-elements, останется рабочий горизонтальный snap-скролл.
