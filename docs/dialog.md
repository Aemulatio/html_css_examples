# HTML dialog: практическое объяснение

## Что это
`<dialog>` — нативный компонент для модальных и немодальных диалогов.

## Базовый пример
```html
<dialog id="modal">
  <p>Сообщение</p>
  <button id="close">Закрыть</button>
</dialog>
```

```js
modal.showModal(); // модальный режим
modal.close();
```

## Когда использовать
- подтверждения действий;
- короткие формы;
- системные сообщения в UI.

## Практические советы
1. Для модалки используй `showModal()`.
2. Добавляй явную кнопку закрытия.
3. Проверяй UX с клавиатурой (`Esc`, `Tab`).
