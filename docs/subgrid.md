# CSS Subgrid: практическое объяснение

## Что это
`subgrid` позволяет дочерней grid-сетке наследовать треки (строки/колонки) родителя.

## Базовый пример
```css
.parent {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}

.parent > article {
  display: grid;
  grid-template-rows: subgrid;
  grid-row: span 3;
}
```

## Когда использовать
- одинаковые вертикальные ритмы у карточек;
- синхронизация строк заголовка/текста/кнопки;
- сложные карточные layout-решения.

## Практические советы
1. Используй `subgrid`, когда нужна именно согласованность треков.
2. Для простых случаев обычного `grid` часто достаточно.
3. Проверяй поддержку в целевых браузерах.
