# CSS Scroll-driven Animations: практическое объяснение

## Что это
Позволяет анимировать элементы по прогрессу скролла, а не по времени.

## Базовый пример
```css
.track {
  scroll-timeline-name: --panel;
  scroll-timeline-axis: block;
}

.progress {
  animation: fill linear both;
  animation-timeline: --panel;
}
```

## Когда использовать
- индикаторы чтения/прогресса;
- привязанные к скроллу эффекты в секциях;
- storytelling интерфейсы.

## Практические советы
1. Не перегружай страницу множеством таких анимаций.
2. Держи fallback без критической зависимости от эффекта.
3. Проверяй производительность на слабых устройствах.
