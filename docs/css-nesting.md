# CSS Nesting: практическое объяснение

## Что это
Нативная вложенность CSS-правил без препроцессоров.

## Базовый пример
```css
.card {
  & h2 { margin-top: 0; }

  & a {
    color: #0f766e;

    &:hover {
      text-decoration-color: #ef4444;
    }
  }
}
```

## Когда использовать
- локальные компонентные стили;
- уменьшение повторяющихся селекторов;
- читаемая структура CSS возле компонента.

## Практические советы
1. Не делай слишком глубокую вложенность.
2. Держи вложенность в пределах 2-3 уровней.
3. Комбинируй с понятными классами, а не со сложными цепочками.
