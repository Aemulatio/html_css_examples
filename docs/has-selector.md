# CSS :has(): практическое объяснение

## Что это
`:has()` — «родительский» селектор: позволяет стилизовать элемент, если внутри него есть нужный дочерний паттерн.

## Базовый пример
```css
.card:has(input:checked) {
  border-color: #0f766e;
  background: #ecfffb;
}
```

## Когда использовать
- подсветка блока при выбранном чекбоксе;
- состояние формы на уровне контейнера;
- компактные интерактивные паттерны без JS.

## Практические советы
1. Используй `:has()` для локальных UI-состояний.
2. Избегай слишком сложных вложенных условий.
3. Держи fallback в уме для старых браузеров.
